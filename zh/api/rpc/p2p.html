# API

# p2p_networkAvailable

现在节点网络是否可用

  • Parameters: none

  • Returns: bool

  • Example:

    {
	"jsonrpc": "2.0",
	"id": 2,
	"method": "p2p_networkAvailable",
	"params": null
}
    {
	"jsonrpc": "2.0",
	"id": 2,
	"result": true
}

    # p2p_peersCount

    当前节点连接的外部节点数量

    • Parameters: none

    • Returns: int

    • Example:

      {
	"jsonrpc": "2.0",
	"id": 3,
	"method": "p2p_peersCount",
	"params": null
}
      {
	"jsonrpc": "2.0",
	"id": 3,
	"result": 10
}
