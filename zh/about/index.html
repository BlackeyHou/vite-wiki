# 简介

Vite Labs

Vite 项目正在开发，有关Vite的相关的技术文档均会放此处，若有什么建议，欢迎提PR。

有关文档内容的修正，请勿直接修改，可以另开分支修改，然后提PR。修改入口在每个页面左下角。

目前技术设计前期的文档请放到 docs/zh/technology文件夹下，docs/zh/technology/README.md为该栏目首页。

本项目使用vuePress，有关侧边栏的配置说明，请移步: vuePress 侧边栏

注意

请确保你的 Node.js 版本 >= 9。

# 运行指南

# 环境要求

  • node: >= 9.xx
  • yarn: >= 1.3.xx

# 安装 yarn

brew install yarn

有关yarn的其他安装说明，请移步： yarn document

# 安装依赖

在doc.vite.org项目路径下，执行以下代码：

yarn

# 开始写作

npm run dev

# Demo 组件

内置Demo组件，用于分Tab展示示例代码。

  • 用法：

    :::demo
``` language tab:TabName
// There is the code  
```  
``` language test:TestCaseName
// There is the code
```

// Test without testname
``` language test
```
:::

  • Example:

    :::demo
``` json tab: Tab1 名称
    {
      "test": 1
    }
```
``` json tab: Tab2 名称
    {
      "result": success
    }
```
``` json test
      {
        "test": 1
      }
```
``` json test: 这是test标题
      {
        "test": 1
      }
```
:::
